# 🎨 Design Token Generator

Generate CSS variables and JSON design tokens from a color palette.

## Features

- CSS Variables
- JSON Export
- Lightweight
- CLI Based

## Run

```bash
python main.py
```

Example

```
Primary
#2563EB

Secondary
#10B981
```

Generated

```css
:root{
--primary:#2563EB;
--secondary:#10B981;
}
```
